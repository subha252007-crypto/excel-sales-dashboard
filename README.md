# Monthly Budget Tracker — Excel

## Overview
A personal budget tracker built in Microsoft Excel to track 
monthly income and expenses by category.

## Tools Used
Microsoft Excel — SUMIFS, IFERROR, Conditional Formatting, Bar Chart

## Key Features
- Category-wise expense breakdown using SUMIFS
- Auto-calculated balance (Income - Expense)
- Conditional formatting highlights overspend above ₹2000
- Bar chart for visual expense comparison

## Summary
- Total Income: ₹30,000
- Total Expense: ₹16,599
- Balance: ₹13,401
- Highest spend: Rent (₹8,000)
